## 🎯 你需要做的（只需一次，5分钟）

### 第一步：打开终端，运行一键部署脚本

```bash
cd ~/Desktop/airport-obstacle-web && bash 部署到云端.sh
```

脚本会自动：
1. 打开浏览器让你登录 GitHub（点击确认即可）
2. 创建 GitHub 仓库
3. 上传所有代码

### 第二步：在 Streamlit Cloud 部署（免费）

1. 打开浏览器访问：**https://share.streamlit.io**
2. 点击 **「Continue with GitHub」** 用你的 GitHub 账号登录
3. 点击 **「New app」**
4. 填写以下信息：
   - Repository: `593682963wd-wq/airport-obstacle-web`
   - Branch: `main`
   - Main file path: `app.py`
   - App URL: 自定义为 `wangdi-obstacle`
5. 点击 **「Deploy!」**

### 完成！🎉

部署成功后，你的网址是：

**👉 https://wangdi-obstacle.streamlit.app**
