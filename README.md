# ✈ 机场障碍物对飞行影响分析系统 — Web 版

Airport Obstacle Impact Analysis System

基于 ICAO Annex 14 / PANS-OPS 标准的机场障碍物分析工具。

## 功能

- 自动解析 PEP TXT / AIP PDF / AIP TXT 格式数据
- ICAO 标准障碍物面评估（1.2% 梯度面 + 保护区包线）
- 遮蔽检测
- 生成 PEP 格式 TXT 输出文件
- 生成 XLSX Excel 分析报表

## 使用方法

1. 上传数据文件（PEP TXT / AIP PDF / AIP TXT）
2. 在「离场参数」中设置转弯角（直飞可跳过）
3. 在「导出」中点击计算，下载结果
